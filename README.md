A sleek, responsive, and feature-rich music player built with **HTML, CSS, and JavaScript**.

🌐 **Live Demo:** [🎵 MusiWave](https://music-player-musi-wave.vercel.app/)

## 📖 About

MusiWave is a modern browser-based music player designed to deliver a smooth and immersive listening experience. It combines an elegant user interface with powerful music management features such as playlists, favorites, lyrics synchronization, theme customization, and music sharing.

Whether you're organizing your favorite tracks or enjoying seamless playback, MusiWave provides a complete music experience directly in your browser.

---

## ✨ Features

### 🎵 Music Playback

* Play, pause, and skip tracks
* Interactive progress bar
* Volume control
* Shuffle mode
* Repeat mode

### ❤️ Favorites

* Mark songs as favorites
* Quick access to liked tracks
* Personalized music collection

### 📂 Playlist Management

* Create custom playlists
* Add or remove songs
* Organize music efficiently

### 🎨 Theme Customization

* Dynamic theme switching
* Modern and responsive UI
* Enhanced user experience

### 📜 Lyrics Support

* Real-time lyrics display
* Sing along with your favorite songs

### 🔗 Sharing & Downloads

* Share songs via direct links
* Download tracks for offline listening

## 🚀 Getting Started

### Prerequisites

* Modern Web Browser

  * Google Chrome
  * Mozilla Firefox
  * Microsoft Edge
  * Safari

* Optional:

  * Visual Studio Code
  * Live Server Extension

---

### Installation

#### Clone Repository

```bash
git clone https://github.com/Shrutika2026/MusiWave.git
```

#### Navigate to Project Folder

```bash
cd MusiWave
```

#### Run Project

**Option 1 (Recommended)**

Open the project in Visual Studio Code and launch it using the Live Server extension.

**Option 2**

Open the `index.html` file directly in your browser.

---

## ☁️ Deployment

MusiWave is deployed on Vercel.

### Deploy Your Own Version

1. Fork the repository
2. Login to Vercel
3. Click **Add New Project**
4. Import the GitHub repository
5. Deploy with default settings

Since MusiWave is a static frontend project, no build configuration is required.
